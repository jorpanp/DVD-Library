# DVD-Library

## kort samanfattning
Mitt projekt är ett program som låter dig hantera en samling DVD-DVD Library är ett Python-projekt som låter dig hantera en samling DVD-filmer genom att lägga till, ta bort och visa filmer. Filmerna sparas i ett JSON-dokument för att säkerställa att samlingen bevaras mellan körningar av programmet.

## Funktioner
- Lägga till en ny DVD-film i biblioteket
- Visa alla DVD-filmer som finns i biblioteket
- Ta bort en DVD-film ur biblioteket

## Lägga till fördefinerade filmer
Följande filmer läggs till i programmet när du kör library.py filen.

- The Matrix
- Inception
- The Godfather
- Pulp fiction
